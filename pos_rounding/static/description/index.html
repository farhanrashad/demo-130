POS Rounding Off Amount

This module allows the seller to round off the price of any order at Point of Sale while payment is made. Also, the price can be round off automatically or manually on the basis of backend configuration . In manual rounding off seller has to click on the particular rounding off payment method to round off the amount. In case of automatic rounding, a rounding off payment line will be automatically added in POS Payment Screen

Features
  • This module allows the seller to manually round off the order amount while payment is made.
  • It also provides the option to automatically round off the order amount.
  • Seller can enable or disable automatic rounding from the backend.
  • This module provides an option to hide or show payment methods in POS Session
  • Journal entries are managed automatically.
  • No complex backend configuration
  • POS Session View
  • Backend Configuration

POS Rounding Off Amount

Manual Rounding

In manual rounding, seller has to click Rounding Payment method for rounding off total amount, i.e. a rounding payment line with rounding amount is added on clicking rounding payment method.

Automatic Rounding

In automatic rounding,on clicking any payment method in which automatic rounding option is checked,a rounding payment line with rounding amount is automatically added.

Option To Auto Update Rounding Amount

If any modification is done in total amount (amount with tax) by adding or removing items from cart then seller can update rounding amount in existing rounding payment line just by clicking 'refresh icon' in rounding payment line. For example: If rounding for order is done and seller go back to product screen to add or remove items from cart then in payment screen if rounding payment line exist then there will be an option (refresh icon) in rounding payment line to auto-update rounding amount on the basis of total amount after adding/removing items from the cart.

Configuration For Showing or Hiding Payment Method In POS Session

Normally Rounding Payment Method will not be shown in POS if Seller wish to show rounding payment in POS then he can check 'Visible In POS to show in POS.

Configuration for Automatic Rounding

Check 'Allow Automatic Rounding' option to allow automatic rounding i.e. whenever user click on this payment method, a rounding payment line with rounding amount will automatically added if no rounding payment line exist for order.
